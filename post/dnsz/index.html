电脑游戏性能优化杂项

2020-03-09 3 min read # 系统优化 # csgo

一、电源计划

首先win10系统版本需要1803以上，打开右击任务栏点击设置-->系统-->关于-->Windows 规格，就可以看到版本号了。

开启步骤如下：

1、使用快捷键Windows+R打开运行界面，输入“powershell“，回车打开。

2、然后输入命令powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61 按回车确定，一会就发现开启了卓越性能。

3、接着依次控制面板-->硬件和声音-->电源选项，可以在隐藏附加计划下面勾选“卓越性能”，关闭设置。

二、鼠标设置

1、打开控制面板，将查看方式改为大图标，找到鼠标。

2、在鼠标属性，指针选项一栏中，将指针移动速度设定在6格，取消提高指针精确度。

3、点击确定完成设置。

三、显卡设置

显卡设置需要下载相应显卡驱动。以N卡为例，A卡对比设置即可。

1、桌面右击，选择NVIDIA 控制面板。

2、在3D设置--通过预览调整图像设置，勾选“使用我的优先选择，侧重于：”，将滑块调至”性能“，点击”应用“保存。

3、在管理3D设置中，可在全局设置中如图调试。也可通过程序设置，选择需要调试的游戏程序，这样仅对游戏调整，点击”应用“保存。

4、在配置 Surround、PhysX中，在PhysX设置--处理器下拉菜单中选择自己的显卡，点击”应用“保存。

四、调整进程优先级

以CSGO为例

1、打开CSGO，使用快捷键Ctrl+Alt+Delete，打开任务管理器。

2、右击csgo.exe，选择转到详细信息，右击csgo.exe，设置优先级，选择“高”。

3、完成设置。

以下是针对CSGO的启动项设置

Steam设置启动项方法：

1.打开steam并点击‘库’

2.右键csgo选择属性

3.常规---设置启动选项

常用启动项说明

-novid 关闭过场动画 -nojoy -d3d9ex 优化帧数 -noforcemparms 游戏时禁用win键
-perfectworld 国服 -worldwide 国际服 -high 优化帧数
-windowed 窗口化 -fullscreen 全屏 -refresh 144 固定刷新率为144
-w 1920 -h 1080 设置分辨率为1920x1080 -tickrate 128 设置128tick -processheap 缓解游戏长时间运行卡顿

注：

​ -tickrate 128 仅单机开服务器用的128tick。新安装的游戏进游戏后可能需要在控制台中输入cl_cmdrate 128 ; cl_updaterate 128 ; rate 786432; 这三个参数才有用。

​ -high 优化帧数就是提高进程优先级，与第四步设置意义相同。

常用国服启动项建议：-perfectworld -novid -nojoy -d3d9ex -high -processheap

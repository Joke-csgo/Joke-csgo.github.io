1、可联网的电脑，并安装有Gridea。

2、Git 安装

3、Gridea与Github的配置

4、使用Gitalk为网站添加评论功能

5、Gridea详细功能介绍

二、Git安装

1、在[Git官网]首页，点击Download 2.25.1 for Windows下载Git安装包。（注：版本不同没有关系）

2、点开安装程序

​ 1）选择安装路径，推荐选择不要带有中文或特殊符号的安装目录。

​ 2）在Select Components界面中推荐勾选如下，然后点击Next>。

​ 3）在Select Start Menu Folder界面，点击Next>。

​ 4）在Choosing the default editor used by Git 界面，点击Next>。

​ 5）在Adjusting your PATH environment界面，选择如图，点击Next>。

​ 6）在Choosing HTTPS transport backend界面，一直点击Next>，直到完成安装。

​ 7）使用快捷键windows+r，输入cmd回车，在弹出的窗口输入git --version，查看是否安装成功。

三、Gridea与Github的配置

1、Github账号注册

​ 1）打开Github注册网站，注册账号。

​ 2）注册完成，登录网页。在网页左侧找到Your most active repositories will appear here. Create a repository or explore repositories.点击蓝色字体Create a repository，进行创建仓库。

​ 3)打开，在Repository name一栏填写如下格式xxxx.github.io，其他如图勾选，点击Create repository。

​ 4）完成仓库创建。

2、设置Github令牌

​ 1）点击右上角头像，点击Settings，左侧菜单中找到Developer settings，点击打开，在左侧菜单中打开Personal access tokens，点击右上角按钮Generate new token。

​ 2）Note栏可随意填写，可填写仓库名称，下面表单勾选repo或者全部勾选也行，点击按钮Generate token

​ 3）复制绿色框中代码至记事本留用，关闭此页面代码将无法显示。

3、配置Gridea

​ 1）如图配置，点击保存，点击检测远程连接。

​ 2）点击文章，勾选两个默认生成的文章“关于”和“Hello Gridea”，点击同步。

​ 3）打开浏览器，输入配置的域名，如https://xxxx.github.io，即可查看配置的主页。
